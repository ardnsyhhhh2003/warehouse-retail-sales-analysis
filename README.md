# Warehouse & Retail Sales Analysis

## Overview
Analisis data penjualan warehouse dan retail periode 2017–2020 menggunakan Python.

## Dataset
Dataset diambil dari Kaggle:
https://www.kaggle.com/datasets/lalit7881/warehouse-and-retail-sales/data

## Tools
- Python (Pandas, Matplotlib, Seaborn)
- Google Colab
- Dataset: Warehouse and Retail Sales (Kaggle)

## Key Insights
- **Beer** adalah kategori produk terlaris dengan total sales 10.2 juta
- Terjadi penurunan drastis di **2018** (~79%) yang perlu investigasi lebih lanjut
- **Retail Sales** lebih dominan dibanding Warehouse Sales di semua tahun
- **Crown Imports** adalah supplier terbesar dengan kontribusi 2.5 juta
